# Product Overview

BeGit は「BeReal × GitHub」コンセプトの開発者向けチーム SNS。リモートチームの「今なに作ってる？」を、義務感なくゲーム感覚で共有させる。

技育CAMP ハッカソン2026 vol.3 出展作品。

## Core Capabilities

- **通知発行 & 1時間チャレンジ**: 1スプリント1人1回、任意タイミングで通知を発行。受け取った全員が1時間以内に投稿すると `On Time` バッジ取得
- **GitHub 自動連携**: 投稿時に GitHub API からコミット数・変更行数・最新コミットメッセージ等を自動取得。ほぼワンタップで投稿完了
- **ぼかしフィード**: 自分が未投稿の間は他メンバーの投稿内容がぼかされる → 投稿意欲を引き出す
- **リポジトリベース自動グループ参加**: GitHub のコラボレーター情報を参照して、BeGit 連携済みユーザーをグループへ自動追加

## Target Use Cases

- リモート・分散チームで「一緒に作ってる感」を出したい
- スタンドアップを義務感なく習慣化したい
- GitHub 活動を自然にチームで可視化したい

## Value Proposition

BeReal の「リアルタイム性 + ぼかし解放」メカニズムを開発現場に移植。`いつ通知を打つか` という戦略性が、義務ではなく参加したくなる仕組みを生み出す。
